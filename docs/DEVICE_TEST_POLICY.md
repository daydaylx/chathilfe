# DEVICE_TEST_POLICY.md

## Entscheidung

Gerätetests werden für diesen MVP gebündelt in Phase 8 durchgeführt.

## Regel

- Phase 3 bis 7 dürfen ohne blockierenden Gerätetest weiterlaufen.
- Code- und Build-Prüfungen sollen vorher soweit möglich erledigt werden.
- Nicht getestete Geräteaspekte bleiben bis Phase 8 als Risiko offen.
- Erfolgreiche Gerätetests dürfen nur genannt werden, wenn sie wirklich durchgeführt wurden.

## Phase 8

Phase 8 ist der Sammelpunkt für die echte Geräteprüfung.

Dort werden APK-Installation, Overlay, WhatsApp-Erkennung, Input-Bar, Result-Panel, Clipboard, Retry und Samsung-Verhalten geprüft.
